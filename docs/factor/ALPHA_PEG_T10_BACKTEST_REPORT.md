## 🎯 策略规则详解

### 1. 因子计算
```
alpha_peg = PE_TTM / 单季净利润增长率
```

### 2. 数据处理
- **数据源**: daily_basic (PE), fina_indicator (净利润增长率)
- **异常值处理**: 分行业3σ原则截断
- **标准化**: 无标准化
- **行业分类**: 使用行业特定阈值

### 3. 选股规则
```
每日选股流程:
1. 按行业(l1_name)分组
2. 计算每只股票的alpha_peg值
3. 分行业异常值处理
4. 分行业排名 (越小越好)
5. 选择每行业前3名
```

### 4. 交易规则
```
持有逻辑:
- 买入: 每日开盘时持有选股
- 持有: 10天 (非交易日顺延)
- 卖出: 持有期满自动卖出
- 资金: 等权分配
```
