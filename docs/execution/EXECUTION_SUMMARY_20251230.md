### 中性化说明

**公式**: `因子残差 = 原始因子 - β1 * log(市值) - β2 * 行业哑变量`

**处理流程**:
1. 对数市值转换：log(流通市值)
2. 行业哑变量编码（One-hot）
3. 线性回归拟合
4. 计算残差
5. Z-Score标准化（均值0，标准差1）

## 🎯 策略特点

### 优势
1. **多维度**: 6个因子覆盖估值、趋势、强度、位置、动量、量能
2. **行业中性**: 剔除行业偏见，跨行业公平比较
3. **市值中性**: 剔除大小盘效应，专注因子选股能力
4. **标准化**: 所有因子标准化为Z-Score，可比性强

### 中性化效果
- **alpha_peg**: R²=1.41%，因子纯净度高
- **alpha_010**: R²=5.58%，轻微受风格影响
- **alpha_038**: R²=9.66%，中等风格暴露
- **alpha_120cq**: R²=15.70%，价格位置因子受市值影响较大
- **cr_qfq**: R²=14.93%，动量因子受市值影响较大

**结论**: 中性化有效剔除了市值和行业暴露，保留了因子的独立选股能力。
