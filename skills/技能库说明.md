## 📚 技能开发

### 创建新技能

使用内置的claude-skills技能：

```bash
# 进入skills目录
cd skills/claude-skills

# 创建新技能骨架
./scripts/create-skill.sh my-new-skill --full --output ..
```

### 验证技能

```bash
# 基础验证
./scripts/validate-skill.sh skills/my-new-skill

# 严格验证
./scripts/validate-skill.sh skills/my-new-skill --strict
```

### 技能规范

每个技能必须包含：
- `SKILL.md` - 技能主文件（带YAML frontmatter）
- `references/` - 详细文档（可选但推荐）
- `scripts/` - 工具脚本（可选）
- `assets/` - 模板资源（可选）
