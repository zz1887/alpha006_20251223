## 核心工具详解

### 1. REPL (JavaScript 运行时)
- 完整 ES6+ 支持
- 预加载 5 个库：
  - D3.js (数据可视化)
  - MathJS (数学计算)
  - Lodash (实用工具)
  - Papaparse (CSV 解析)
  - SheetJS (Excel 处理)
- 异步支持 (async/await)
- BigInt 支持
- WebAssembly 支持
- 文件读取能力

### 2. Artifacts (可视化输出)
- React 组件
- Three.js 3D 渲染
- HTML/SVG 生成
- 图表和可视化
- 交互式界面

### 3. Web Search (网络搜索)
- 搜索网络内容
- 域名过滤
- 仅美国可用

### 4. Web Fetch (内容获取)
- 获取网页内容
- HTML 转 Markdown
- 内容提取

### 5. Conversation Search (对话搜索)
- 搜索历史对话
- 上下文检索

### 6. Recent Chats (最近对话)
- 访问最近会话
- 对话历史管理

### 7. End Conversation (结束对话)
- 会话清理
- 对话总结
